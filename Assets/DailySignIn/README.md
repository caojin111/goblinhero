## 文件命名说明：
- 所有图片文件请使用 PNG 格式
- 文件名必须与代码中引用的名称完全一致
- 将所有文件放入 `/Users/lazyg/Documents/goblinhero/Assets/DailySignIn/` 文件夹

## Xcode 配置：
1. 在 Xcode 中右键点击 `Assets` 文件夹
2. 选择 "Add Files to A004..."
3. 选择所有下载的图片文件
4. 确保勾选 "Copy items if needed" 和 "Add to targets: A004"
